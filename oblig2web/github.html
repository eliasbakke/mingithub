Git

Git noe du bruker for å kjøre kodedeling og versjonskontroll. Et eksempel er en heis hvor du starter i første etasje. For hver endring du gjør i koden, går du opp en etasje. Hver gang du har lyst å gå tilbake til en viss endring, så tar du bare heisen ned til den etasjen du gjorde den endringen i. Git gjør samarbeid bra og unngår konflikter ved blanding av samme kode på forskjellige maskiner.

GitHub

GitHub er nesten som Git, bare web-basert. Hvis du lager deg en konto på GitHub, kan du legge inn dine filer i et repository og kunne endre på de når som helst.
